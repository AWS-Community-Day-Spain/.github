# AWS Community Day Spain

Comunidad oficial que organiza el **AWS Community Day Spain**, el mayor evento gratuito sobre Amazon Web Services organizado *por* y *para* la comunidad técnica en España.

Cada año reunimos a profesionales, AWS Heroes, Community Builders y User Groups locales para compartir conocimiento, experiencias y casos reales sobre la nube de AWS.

## Web oficial

**[awscommunity.es](https://awscommunity.es)** — siempre apunta a la edición en curso. Cada edición conserva además su propia web bajo el subdominio del año: `<año>.awscommunity.es`.

## Cómo participar

- **Asistir**: las entradas son gratuitas y se anuncian en la web del año.
- **Ponentes**: abrimos un Call for Papers cada edición.
- **Patrocinadores**: contacta con nosotros si quieres apoyar al evento.
- **User Groups locales**: súmate al grupo AWS de tu ciudad.
